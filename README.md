# app

## Project setup
```
docker-compose build

docker-compose run --rm --no-deps app npm i
```
(Some warns are expected)

### Run App
```
docker-compose up
```
App will be running on http://localhost:3000

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
